# ARC v1.0 Old Archive

Historical archive of the old ARC Android robot/controller project.

## ARCv0.3_RX2_generated_sound

Type: generated sound

Contains:
- `ARC v0.3 RX2.apk`
- Java Android source: `ARC-0.3`
- command-generation notes: `Commands generator.txt`

This version generates command audio using Android `AudioTrack` / `WAVGen`.

## ARCv09_Tank_audio_file_based

Type: audio-file based

Contains:
- `ARCv09 Tank.apk.rar`
- Python SL4A source: `source/sl4a/ARCv09.py`
- WAV command audio files in `audio/`

## ARCv09_Car_audio_file_based

Type: audio-file based

Contains:
- `ARCv09 Car.apk.rar`
- Python SL4A source: `source/sl4a/ARCv09.py`
- WAV command audio files in `audio/`

## Instructions

English instruction:
- `Instruction_EN_fixed.docx`

Original Russian instruction:
- `Instruction_RU_original.docx`

## Notes

`ARC-0.3` is kept as the best source match for `ARC v0.3 RX2.apk`.

Tank and Car versions are audio-file based because they use Python SL4A and WAV command files.
